## What is Cobar?

Cobar is a proxy for sharding databases and tables,compatible with MySQL protocal and MySQL SQL grama,underlying storage only support MySQL for support foreground business more simple,stable,efficient and safetyã€‚

- __Sharding__
You can add new MySQL instance as your business grows.

- __High availability__
Both Cobar server and underlying MySQL is clustered,business will not suffer with single node fail.

- __Compatible with MySQL protocol__
Use Cobar as MySQL. You can replace MySQL with Cobar to power your application.
